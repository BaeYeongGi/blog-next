## 02_컴파일러_인터프리터

* ※ 실행순서
  * a) 전처리기: 전처리 구문을 처리하는 과정(.c)
  * b) 컴파일러: 고수준 → 저수준 언어로 변환(기계어와 가까운 형태의 언어)(.i)
  * c) 어셈블러: 완전한 기계어로 바꾸어주는 역할(.o)
  * d) 링커: 여러 개의 오브젝트 파일 합치거나 or 라이브러리를 합치는 역할(.exe)

* ※ 컴파일러: 인간 고급언어 → 컴퓨터 저급언어로 번역
  * └ 과정: 전처리기 → 컴파일러 → 어셈블러 → 링커
  * └ 종류: C, C++, Java
  * └ 번역시간 오래걸리지만, 한 번 번역 후 다시 번역하지 않으므로 실행속도 빠름 

* ※ 인터프리터: 고급언어로 작성된 프로그램을 한 줄씩 읽어서 실행하는 프로그램(통역사라고 이해)
  * └ 종류: Ruby, php, javascript, Basic
  * └ 번역 & 실행이 동시에 이루어짐 → 별도의 실행파일이 존재하지 않음

## 03_변수
- 정의: 데이터를 저장할 수 있는 메모리의 기억 공간<br/>

- ※ 변수생성 규칙 및 특징
  * a) 예약어(키워드)는 사용할 수 없다 (if.. break… else…)
  * b) 공백을 포함할 수 없다
  * c) 첫 글자는 영문과 언더바(_)만 사용이 가능하다.
  * d) 언더바(_) 이외의 특수문자는 사용할 수 없다.
  * e) 대소문자를 구분한다

- ※ 데이터 타입(자료형)
  * 문자형: char 1Byte
  * 정수형: short 2Byte / int 4Byte / long 4Byte
  * 실수형: float 4Byte / double 8Byte

## 04_변수의 종류와 유효범위

- ※ 지역변수
  * a) 함수/블록 안에서 선언하는 변수
  * b) 함수/블록의 종료와 함께 메모리상에서 소멸
  * c) 초기값이 없으면 쓰레기 값이 들어감
  * d) 스택 영역에 저장됨

```c title="지역변수 예제"
void main(){
  if(1){ // 1은 참 (true)
    int num1 = 10;
    printf("%d",num1);
  }
  printf("%d",num1);
}

// int num1 이라는 변수는 if 조건문 안에서만 유효하므로 마지막으로 출력된 printf 는 에러
```

- ※ 전역변수
  * a) 함수 블록 밖에서 선언하며 전체 함수에 영향을 미침
  * b) 프로그램이 종료될 때 메모리상에서 소멸
  * c) 초기값이 없으면 0값이 들어감
  * d) 데이터 영역에 저장됨

```c title="전역변수 예제"
int num1;
void main(){
  num1 = 10;
  printf("%d",num1);
  func();
  printf("%d",num1);
}
void func(){
  num1 = 20;
}
// main() 함수에서 출력되는 printf 값은 10, 20
```

- ※ 정적변수
  * a) static 으로 선언하며 선언된 블록 내에서만 사용가능
  * b) 함수가 종료되어도 소멸되지 않고, 값이 그대로 유지됨
  * c) 프로그램이 종료될 때 메모리상에서 소멸
  * d) 초기값이 없으면 0이 입력되고, 단 한번만 초기화
  * e) 데이터 영역에 저장됨

```c title="정적변수 예제"
void main(){
  add(10)
  add(5)
  add(3)
}

int add(int i){
  static int sum = 0;
  sum = sum + i;
  printf("sum:%d\n", sum);
}

// 인자값으로 10 전달
// 0 = 0 + 10;
// sum: 10 (enter) 메모리상에 남아있음
// 10 = 10 + 5;
// sum: 15 (enter)  메모리상에 남아있음
// 15 = 15 + 3; 
// sum: 18 (enter)  메모리상에 남아있음
// 다음 수행할 함수가 없어 프로그램 종료
// static 정적변수가 아닌 지역변수가 들어가는 경우 10, 5, 3 이 출력됨
```

- ※ 동적변수
  * a) 메모리 동적할당으로 생성되는 변수
  * b) Run time 시간에 메모리를 할당
  * c) 동적 기억장소에 할당하여 메모리 낭비를 피하려는 목적
  * d) 무조건 포인터를 사용해야 함
  * e) 힙 영역에 저장됨

## 05_산술연산자

|구분|종류|예|
|---------|------|------|
|__이항 연산자__|+, -, *, /, %|3 * 4|
|__단항 연산자__|++, --|4++;| 
|__대입 연산자__|=, +=, -=, *=, /=, %=|a += 3;|


```c title="산술연산자 예시"
1) 3/4 = 0
// 정수 나누기 정수는 0
2) a = 10; 
     b = a++;
     c = ++a;
// b의 경우 a라는 값을 b에 넣고 ++ 식을 계산 → 11
// c의 경우 a를 ++ 먼저 증가시키고(12) c에다 대입 → 12
3) a = 10;
     a += 10;
// a = a + 10 → a에 10을 대입 후 계산하면 → 20 
```

```c title="아래 C 프로그램의 실행 결과는?"
#include<stdio.h>
void main(){
  int a, b;
  a = 4 * (1 / 2);
  b = a++;
  printf("%d", b);
}
// a = 4 * 0 → 0 
// b = 0
// a++ → a = 1
// 답: 0
```

## 06_연산자 우선순위

- ※ 산술연산자 종류

|구분|종류|예|
|--------|----|------|
|__높음__| () | 괄호 |
|        | ++, --, ~, ! | 단항 |
|        | *, /, % | 산술 |
|        | +, - | 산술 |
|        | ≪, ≫ | 비트 |
|        | ＜, ＜=, ＞, ＞= | 비트 |
|        | ==, != | 관계 |
|        | &, ^, | 비트 |
|        | && | 논리  |
|__낮음__| =, +=, -=, *=, /=, %= | 대입 |

```c title="연산자 우선순위 예시 1"
int a = 3;
int b = 4;
int c = 5;
int d = 6;

a*b+c>=d&&d/a-b!=0 =

// a*b , d/a 를 먼저 계산 → 12, 2
// 12+c , 2-b 를 계산 → 17, -2
// 17 >= d , -2 != 0 을 계산 → 1, 1
// 1 && 1 = 1
```
```c title="연산자 우선순위 예시 2"
int a = 3;
int b = 4;
int c = 5;
int d = 6;

d%b+ ++a * c-- =  

// ++a , c-- → int a = 4;
// d%b , a * c → 2, 20
// a * c 식을 수행한 후 int c = 4; 가 됨
// 2 + 20 = 22
```
```c title="아래 C프로그램의 출력은?"
#include<stdio.h>
int main(){
  int a = 5, b = 5;
  a *= 3 + b++;
  printf("%d %d", a, b);
  return 0; 
}
// 3 + b++ = 8, int b = 6;
// a = a * 8 → 40
// a = 40, b = 6
```

## 07_관계_논리연산자_조건문

- ※ 관계연산자 종류

|연산자|예|결과값|
|------|--|-----|
|__==__| 10==9 | 0 |
|__!=__| 3!=3 | 0 |
|__＞__| 5＞5 | 0 |
|__＞=__| 5＞=5 | 1 |
|__＜__| 7＜7 | 0 |
|__＜=__| 7＜=7 | 1 |

- ※ 논리연산자 종류류

|연산자|예|결과값|
|------|--|-----|
|__!__| !9 | 0 |
|__&&__| 3＞=3 && 3＞3 | 0 |
|__┃┃__| 5＞5 ┃┃ 5＞=5 | 1 |

```c title="다중 if문"
if(조건1){
  조건1이 만족할때의 처리
}
else if(조건2){
  조건 2가 만족할때의 처리
}
else {
  모든 조건이 아닐때의 처리
}

// 조건1의 경우 만족하면 나머지 조건2, else는 수행하지 않음
// 조건1을 만족하지 못하면 조건2를 수행, 
// 조건1, 2를 만족하지 못하면 else 수행
```

```c title="아래 C프로그램의 출력은?"
#include<stdio.h>
int main(){
  int i = 3;
  int j = 4;
  if((++i > j--) && (i++ < --j)) i = i-- + ++j;
  else j = i-- - --j;
  printf("%d\n", i);
}

// if 조건문 중 괄호안에 있는 계산식부터 수행 
// ++i > j-- → int i = 4; → 4 > 4; → int j = 3; 
// 위의 계산은 거짓이므로 0
// && 연산을 더 수행할 필요가 없으므로 else 로 이동 (i = 4; j = 3;)
// else 수행 j = 4 - 2 → int i = 3; → j = 2; 
// 답: 3 
```

## 08_논리연산자_증감연산

- ※ 관계연산자 종류

|연산자|예|결과값|
|------|--|-----|
|__==__| 10 == 9| 0 |
|__!=__| 3 != 3 | 0 |
|__＞__| 5 ＞ 5 | 0 |
|__＞=__| 5 ＞= 5 | 1 |
|__＜__| 7 ＜ 7 | 0 |
|__＜=__| 7 ＜= 7 | 1 |

```c title="아래 C프로그램의 실행결과는?"
#include<stdio.h>
int main(){
  int a = 3+5, b=1, c;
  int ap, bp;
  ap = a++;
  bp = ++b;
  b = 3 * (ap == 8);
  c = 5 * (ap != 8);
  printf("%d %d %d %d %d", a, b, c, ap, bp);
}

// a = 8 → ap = a++ → int a = 9; / int ap = 8;
// bp = ++1; → bp = 2; / int b = 2; 
// b = 3 * (8 == 8); → int b = 3;
// c = 5 * (8 != 8); → int c = 0;
// 답: a = 9; b = 3; c = 0; ap = 8; bp = 2;
```

## 10_출력형식

- ※ 출력 변환기호

|기호|의미|데이터 타입|
|------|----------------------|-----------|
|__%d__| 부호 있는 10진수 출력 | 정수, 문자 |
|__%f__| 고정 소수점으로 출력 | 정수, 실수 |
|__%c__| 문자 출력 (1자) | 문자 |
|__%s__| 문자열 출력 | 문자열 포인터 |
|__%x__| 16진수 출력 | 정수 |
|__%o__| 8진수 출력 | 정수 |

```c title="아래의 실행결과는?"
printf("값1: %d, 값2: %d", 10, 5); 
// 값1: 10, 값2: 5
printf("값: %c", 65);
// 값: A
printf("값: %c", 'A');
// 값: A
printf("값: %s", "안녕하세요.");
// 값: 안녕하세요.
printf("값: %x", 34); 
// 64 | 32 | 16 | 8 | 4 | 2 | 1 
//  0 |  1 |  0 | 0 | 0 | 1 | 0
// 값: 22 (16진수는 4개씩 끊어서) 
printf("값: %o", 34);
// 값: 42 (8진수는 3개씩 끊어서)
```

- ※ 정수형 자리수 지정
  * a) printf("값: %d", 13); [1][3]
  * b) printf("값: %5d", 13); [ ][ ][ ][1][3]
  * c) printf("값: %05d", 13); [0][0][0][1][3]

```c title="아래의 실행결과는?"
printf("값: %f", 3.26);
// 값: [3][.][2][6][0][0][0][0]
// 소수점 6자리까지 출력
printf("값: %6.1f", 3.3);
// 값: [][][][3][.][3]
// 6개 공간 확보 후 소수점 1자리까지 표현
printf("값: %-6.1f", 3.26)
// 값: [3][.][3][][][]
// 음수는 앞에서부터 출력 & 소수점 1자리까지 표현 & 반올림
printf("값: %-6.3f", 3.26)
// 값: [3][.][2][6][0][]
// 음수는 앞에서부터 출력 & 소수점 3자리까지 표현 & 반올림
```
```c title="아래의 실행결과는?"
#include<stdio.h>
void main(){
  double d_value;
  float f_value = 5.65;
  int n;
  d_value = f_value;
  d_value = d_value + 0.5;
  n = (int)d_value;
  printf("%d3.1f, %d", d_value, n);
}

// d_value = 6.15
// n = 정수형으로 변환(소수점 버림) → 6
// printf → 3개의 공간에서 소수점 1자리까지 표현 & 반올림
// 6.2, 6 
```
