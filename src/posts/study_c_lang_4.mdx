## 31_포인터 배열_1

※ 배열의 요소가 포인터(메모리주소)로 이루어진 것

```c title="다음 C 프로그램의 실행 결과는?"
#include<stdio.h>
int main(){
  static char *c[] = {"aaa", "bbb", "ccc"};
  printf("%s", *(c+1));
}
// c 100 주소값이 들어가는 배열 
// 100번지 [20번지], 101번지 [30번지], 102번지 [40번지]
// 20번지 a a a
// 30번지 b b b
// 40번지 c c c

// *(c+1) → 101번지의 값 → 30번지의 값
// 답: bbb 

```

## 33_포인터 배열_3

```c title="다음 C 프로그램의 실행 결과는?"
void main(){
  int a[2][3] {{-3, 14, 5}, {1, -10, 8}}; // a[0], a[1] 에 해당
  int *b[] = {a[0], a[1]};
  int *p = b[1];
  printf("%d", *b[1]);
  printf("%d", *(++p));
  printf("%d", *(--p-2));
}

// int a 2개의 배열 공간 

// a 100번지
// 100 a[0] -3 14 5 
// 103 a[1] 1 -10 8
// *b[] 포인터 변수에는 a[0], a[1] → 100번지, 103번지 저장
// *p 포인터 변수에는 b[1] → a[1]에 해당하므로 103번지 저장

// *b[1] = 1 
// *(++p) → ++p → ++103번지 → 104번지의 값 → -10 
// *(--p-2) → 104 에서 -- → 103 - 2 → 101번지의 값 → 14
// 답: 1, -10, 14
```

## 34_포인터 배열_4

```c title="다음 C 프로그램의 실행 결과는?"
int main(){
  int *arr[3];
  int a = 12, b = 24, c = 36;
  arr[0] = &a;
  arr[1] = &b;
  arr[2] = &c;
  
  printf("%d\n", *arr[1] + **arr + 1);
}
// *arr 주소값 100
// a[0] a의 주소값 200 / 값 12
// a[1] b의 주소값 300 / 값 24
// a[2] c의 주소값 400 / 값 36
// **arr arr의 값의 값
// 100번지의 값의 값 200번지에 해당 → 200번지의 값 → 12
// *arr[1] → 24 + 12 + 1 
// 답: 37
```
